# Extra Storage Add-on

## Overview
The **Extra Storage Add-on** provides additional cloud storage in 50 GB increments for documents, files, and backups. Available for all plan subscribers who need more space than their base plan provides.

## Who Is It For?
- Teams that store large files (design assets, videos, large documents)
- Companies that need to archive historical project data
- Users who have exceeded their plan's storage limit

## Key Features
- **50 GB Additional Storage**: Each unit adds 50 GB to your total storage capacity.
- **Automatic Backups**: All files are automatically backed up with point-in-time recovery.
- **File Versioning**: Previous versions of files are kept for 90 days, allowing you to restore earlier versions.
- **Secure File Sharing**: Share files with external collaborators via secure, expiring links.

## What Problems Does It Solve?
- Prevents running out of storage space without needing to upgrade your entire plan
- Provides a flexible, pay-as-you-grow storage solution
- Keeps historical data accessible rather than requiring manual archiving

## Pricing
- **$9.99/month** per 50 GB block (flat fee, not per-user)
- Stackable: purchase multiple blocks as needed

- Works with **Starter** and **Professional** plans
- **Enterprise** plan includes unlimited storage — this add-on is not needed
